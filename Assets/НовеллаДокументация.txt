Использование Интерактива
Интерактив(TextQuest, DragText)
Интерактив находится между блоками(Соединять их не надо), в скрипте указывается название блока, который вызывается при выполнении интерактива
Для запуска скриптов использовать Scripting/SetActive, в объект прокинуть объект со скриптом и все его дочерние и поставить галочку, не забывать выключать потом
	TextQuest(Перестановка букв)
Right Word - правильное слово
Replacement - перемещение сиволов( (Element 0 - 2) = 0 элемент на позицию 2 и наоборот)
Text Parent - дочерний объект, должен иметь Grid Layout Group для корректной работы
TextPrefab - настроеный элемент текста
BlockName - имя запускаемого блока, при успешном выполнении интерактива
	FindItems(Поиск объектов)
Flowchart - обхект flowchart(он у нас один)
Block Name - Имя вызываемого блока
Чтобы использовать скрипт на областях клика вызвать событие IPoinerClick(Или типа того, для этого добавить EventTrigger), туда прикрепить 
ссылку на объект с этим скриптом и в нем вызвать метод FindItem(int index) index - значение переменной IndexOfItem
И далее уже в зависимости от ее значения выбирать блоки(реплики)
